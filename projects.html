Delivr

Technology Used - Swift, SwiftUI, SwiftyJSON, Aftership

Delivr was built as my final Year Project in my Bachelor of CS. Originally starting out as a fun little project to work on to teach myself Swift, Delivr ended up being made into a full fledged app.

With the ability to add packages from all the major carriers throughout Canada, Delivr provides the functionality to track a users package throughout Canada. By integrating Delivr with Apple Maps, Delivr allows to see the exact route their package took while in transit by showcasing the last update from each loaction the package has been at.

Boasting a beautiful design, inspired by Apples own stock apps such as Mail, Delivr comes as close to a stock package tracking app as possible.

Liquor Locator

Technology Used - Java, Springboot, Maven, Couchbase, SCSS, Vaadin

A group project done to show the location of all establishments with a valid liquor license in Nova Scotia. Using this searchable a user can find a place by name, or establishment type (clubs, pubs, restaurant, etc.).

My main role in this project was to be the head UI designer. I was responsible for creating all related design documents, creating mockups for prototyping, implementing our main UI and implementing a dark mode for night time users.

Having never used SCSS or Vaadin before I got to learn how to use mixins to create dynamic themes for the site.
