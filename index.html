Delivr

Technology Used - Swift, SwiftUI, SwiftyJSON, Aftership

Delivr was built as my final year project during my Bachelor of CS. Originally starting out as a fun project to teach myself Swift, Delivr ended up being made into a full-fledged app.

With the ability to add packages from all the major carriers throughout Canada, Delivr provides the functionality to track a users package throughout the country. By integrating Delivr with Apple Maps, the app allows each user to see the exact route their package took while in transit by showcasing the most recent update from each location the package has been.

Boasting a beautiful design inspired by Apples' own stock apps such as Mail, Delivr comes as close to a stock package tracking app as possible.
